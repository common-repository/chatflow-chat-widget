Add the ability for your visitor to start chat with you on Facebook Messenger and WhatsApp directly from your website.

== Description ==

ChatFlow.io Click To Chat Widget allows your visitors to start chat with you on multiple Messaging Apps easily so you can still keep in touch with them even when they left your website.

Boost your website conversion today!

[Documentation](https://chatflow.io)

= Features =

- Automated Message Triggering
- Call to Action
- Multiple Messaging Apps Support (Facebook Messenger, WhatsApp, Email & Phone Call)
- Customisable Widget Appearance

== Installation ==

= Automatic installation =

1. Go to Plugins > Add New > Upload and select the .zip file from your hard disk
2. Click the "Install now" button
3. Activate the plugin through the 'Plugins' menu in WordPress

= Manual installation =

1. Upload the plugin folder to the `/wp-content/plugins/` directory via ftp
2. Activate the plugin through the 'Plugins' menu in WordPress
 
== Frequently Asked Questions ==

= What are the messaging apps supported? =
ChatFlow.io can support multiple apps: Facebook Messenger, WhatsApp, Email and Phone.

= Can I change the widget position? =
Yes, the widget can be configured to show on the left or right side. X-Axis and Y-Axis Padding can also be added via custom options.

= How do I view the analytics (PRO) =
Once Google Analytics is installed on your website, the widget will automatically create events when user interacts with the widget. No setting is required to enable this.
